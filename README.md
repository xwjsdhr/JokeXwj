# JokeXwj
作者：夏伟佳
时间：2016年2月13日19:23:32  
## 使用到的第三方框架
1.Ion,Picasso 图片加载。 
2.OkHttp 网络请求。 
3.ButterKnife 依赖注入。 
4.Gson Json数据解析。
5.android-gif-drawable Gif图片的显示。
